# 🍅 CLI Pomodoro Timer

Простой и эффективный консольный таймер Помидоро на Python, созданный для контроля времени во время учёбы, кодинга и работы. Помогает поддерживать фокус и не забывать вовремя отдыхать.

## ✨ Особенности

* **Наглядный прогресс-бар**: В реальном времени показывает, сколько процентов текущей фазы уже прошло.
* **Цветовое оформление**: Визуально разделяет рабочее время и отдых с помощью встроенных ANSI-цветов в терминале.
* **Звуковые уведомления**: Подает системный звуковой сигнал (`bell`) при смене фаз.
* **Кастомизация**: Настраиваемая длительность сессий работы (по умолчанию 40 мин), отдыха (10 мин) и количества циклов (4 цикла).
* **Безопасное прерывание**: Корректно завершает работу по сочетанию клавиш `Ctrl + C` без вывода системных ошибок.

## 🚀 Запуск проекта

### Требования
Для работы скрипта необходим установленный **Python 3.x**.

### Инструкция
1. Склонируйте репозиторий или скачайте файл со скриптом.
2. Откройте терминал в папке с файлом.
3. Запустите скрипт командой:
   ```bash
   python main.py
   ```
   *(Замените `main.py` на имя вашего файла, если оно отличается)*

## 🛠️ Как это работает

1. Скрипт запускает приветственное окно и ожидает нажатия `Enter`.
2. Начинается фаза **«УЧЕБА/КОДИНГ»** (40 минут). Экран консоли динамически обновляется каждую секунду.
3. После завершения звучит сигнал, и скрипт предлагает перейти к фазе **«ОТДЫХ»** (10 минут).
4. Процесс циклически повторяется 4 раза, после чего выводится финальное поздравление.
